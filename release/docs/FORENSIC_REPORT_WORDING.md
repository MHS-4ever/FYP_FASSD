# Forensic Report Wording (Phase 9A)

## Recommended Wording

- "evidence indicator"
- "experimental prototype"
- "candidate segment"
- "manual review recommended"
- "not final forensic proof"

## Forbidden Wording

- "production-ready forensic verdict"
- "court-ready automatic proof"
- "final legal determination by model"

## Forbidden Fields

The following fields are forbidden in release source/config outputs:

- `fake_score`
- `real_score`

These terms may appear only in this forbidden section as examples of disallowed fields.
